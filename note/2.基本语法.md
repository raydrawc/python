### 序列类型

列表,元组,集合  
python中**序列类型**包括,**字符串**,**列表**,**元组**,所谓序列类型,则成员有序排列,并且可以通过**下标**偏移量访问, **正序**从0开始,到n-1结束, **逆序**从-1开始,到-n结束  

```python
a = [1,2,3,4,5]
1 == a[0]

```

#### 列表 list

[1,2,3,4,5] 可修改 可通过下标访问

#### 元组 tuple

(1,2,3,4,5) 不可变列表

#### 集合 set

一系列互不相等元素的集合,无序的,非重复的  
{data1, data2}

### 映射类型

#### 字典 dict

{Key1:Val1, Key2:Val2} 键值对类型 没有明确的位置关系,内部无序  
**映射类型** 属于`{k-v}`形式的数据,根据key映射val
